शुरुआत

Various books on type design feature a segment on the anatomy of type, but mostly all books focus on anatomy for the Latin script alone. Majority of the work describing the anatomy for Indian scripts stays with people having in-depth knowledge about the script calligraphy. For beginners in type design such information stays in research papers. Though in recent years, visual guides as well as in-depth text based guides have been made available online.

An interactive visual guide is hard to find, and that’s what I took up for my design project this semester.

Mini-goal: learn about the anatomy of देवनागरी type.

प्रक्रिया

  • Learning about the anatomy of devanagari type.
  • User survey for understanding the need for a guide.
  • Ideation on key features.

अधिगम

To learn about the anatomy, I went through various articles, papers and books on the topic. Mostly all information available through these comprehensive resources are available in text form.

  • History of Devanagari Letterforms by Chitra Gohad
  • Anatomy of Devanagari Typefaces by Prof. Girish Dalvi
  • Devanagari Search Tool by Prof. Girish Dalvi
  • Design parameters of Devanagari by Mukund Vasudeo Gokhale
  • Typography of Devanagari by Bapurao S. Naik
  • Devanagari Type Anatomy by Pooja Saxena

सर्वेक्षण

A user survey was conducted to gain objective insights on the following questions:

  • Are people taught about the anatomy of the script?
  • Do people outside of the design field care about knowing such information?
  • What could be a possible medium for a visual guide?
  • In which way would people understand the information more efficiently?

135 responses

“I am not a designer”: 40%

Key Takeaways

  • There is a general interest to know more about the script.
  • Giving both an overview and an in-depth visual would help understand things better.
  • Interactivity is preferred by most prospective users

सक्रिय अध्ययन

For an interactive learning experience:

  • Two-way communication (feedback to action).
  • Control over source material (choice of what to learn).
  • Interactive effort.
  • Flexible evaluation.

विशेषताएं

Explore

An open interactive space to hover and click over different parts of a given word set. Clicking provides more information about the specific part the learner wants to know more about.

Guided

A curated curriculum for learning about the anatomy from introduction to the script to ways of drawing and proportions to be kept in mind while designing.

Playroom

This interactive space provides options to customise a given character set according to various visual features and proportions including contrast, width, neck height, loops, knots and joins.

Vocabulary

A knowledge base of all the terms related to the anatomy of the script with options to search for specific terms and going through a complete list of terms. each item contains textual and visual information.

साध्यता

Platform and languages

  • Cross-platform availability for web and native application for Windows, macOS, Linux.
  • HTML, CSS, JavaScript, Python and their respective extended libraries.

Explore

  • Developed in Processing’s Python library as higher graphical computation is required which JavaScript will not be able to take full advantage of within a browser.

Playroom

  • Designing all possible combination of characters and using the CSS property font-feature-settings and newer OpenType features that come with the CSS Font Module Level 4 (Working Draft, November 2020 but not fully implemented)
  • OR developing a Python based graphical engine that renders vectors on the go.
